```
cd Bingo

```

## 設計

```
・ 1から１００までのランダムな数値を配置。（数値被りなし）
・ startボタンを押し、stopボタンを押すと、1から100の数値がランダムに選ばれる。（数値被りなし）
・ 縦、横が揃えばBingoとなり、ゲーム終了。
・ resetボタンを押すと、ゲーム再起動。
```
